    typecho本身 #

    typecho编辑器的工具栏在浏览器窗口过窄时，比如用手机打开，就会出现按钮“丢失”。工具栏的id和class都为wmd-button-row，问题出在css文件’admin/css/style.css’中高度的设置为26px，改成auto就可以了。

    .wmd-button-row{list-style:none;margin:0;padding:0;height:26px;line-height:1;}

    原来的CSS样式

    修改过后的工具栏在手机上也可以使用了。
