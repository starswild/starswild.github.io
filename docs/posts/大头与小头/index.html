    大头与小头

    October 18, 2017

    Little-Endian&Big-Endian

    例子 #

    整数：6699 如果左边是起始地址0，右边无限大。 大头：数据从左到右存放->1A2B 小头：数据从右到左存放->2B1A

    代码 # 

    int main(void)  
{  
    int i = 0x12345678;  
    if(*((char*)&i) == 0x12)  
        printf("大端");  
    else  
        printf("小端");  
    return 0;  
}
