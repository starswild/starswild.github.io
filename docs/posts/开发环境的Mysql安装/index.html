    开发环境的Mysql安装

    March 16, 2018

    1. 初始化 mysql –initialize
    2. 配置文件 [client] default-character-set=utf8 [mysqld] port=3306 default-storage-engine=INNODB character-set-server=utf8 collation-server=utf8_general_ci
    3. 启动 mysql –console –explict_default_for_timestamp=true
    4. 连接 mysql -uroot -p
    5. 管理 mysqladmin -uroot -p password mysqladmin -uroot shutdown
